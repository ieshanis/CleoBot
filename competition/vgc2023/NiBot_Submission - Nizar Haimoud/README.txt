to run my bot do the following instructions:

1- Navigate to the directory where the "NiRemoteCompetitor.py" file is located in a Terminal.

2- Run the following command to run NiBot agent:

	python NiRemoteCompetitor.py --id <agent_id>

Replace <agent_id> with the desired ID for the agent. The agent will listen on the remote port "5000 + <agent_id>.

Example: python NiRemoteCompetitor.py --id 12
